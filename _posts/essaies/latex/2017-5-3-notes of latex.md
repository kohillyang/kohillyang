### texlive2016 

```
apt-get install ttf-freefont
sudo mkfontscale
sudo mkfontdir
sudo fc-cache -fv
```

 please consider supporting it by joining the group best for you. The

 Add /usr/local/texlive/2016/texmf-dist/doc/info to INFOPATH.
